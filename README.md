# PerformanceIQ — Student Performance Analytics Dashboard

PerformanceIQ is a web-based analytics dashboard designed to help students and educators monitor academic performance effectively. It calculates SGPA and CGPA, visualizes performance trends through charts, and provides personalized motivational insights.

## Features

* SGPA Auto Calculation
* CGPA Tracking
* Student Performance Analytics
* Department Comparison Charts
* Search and Filter Students
* Dark Mode UI
* Personalized Motivational Insights

## Tech Stack

* HTML
* CSS
* JavaScript
* Chart.js

## Purpose

I built this tool to simplify academic performance tracking and visualize improvement over semesters.
